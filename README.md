# Expense_Tracker
💰 Personal Finance Dashboard

A modern and interactive Personal Finance Dashboard that allows users to securely track their income, expenses, and balance with real-time updates and visual insights.

This project demonstrates frontend development skills including authentication flow, state management, UI design, and data visualization.


🚀 Features

💳 Finance Management

- ➕ Add income and expense transactions
- 📊 Real-time balance calculation
- 🧾 Transaction history table
- 📈 Interactive pie chart visualization

🔐 User Authentication

- Signup & Login functionality
- User-specific data storage
- Secure session using LocalStorage

🎨 UI & Experience

- 🌙 Dark mode / ☀️ Light mode toggle
- 📱 Clean dashboard layout
- ⚡ Instant UI updates

📄 Reports

- Export transaction history as PDF report


🛠️ Tech Stack

- HTML5 → Structure
- CSS3 → Styling & Themes
- JavaScript (Vanilla JS) → Application logic
- Chart.js → Data visualization
